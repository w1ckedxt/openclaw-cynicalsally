## What Sally Does

- **Chat** — She remembers you. She has opinions. She's sharp, witty, and never boring.
- **Roast** — URLs, images, documents, PDFs. Everything gets the truth.
- **Code Review** — Quick reviews or deep analysis of your code.
- **Full Truth** — Comprehensive deep-dive analysis with a PDF magazine report.

## Install

Set your device ID in OpenClaw config:

```
SALLY_DEVICE_ID=<any-uuid-v4>
```

Then just talk to Sally. She'll take it from there.

## SuperClub

Free users get 3 roasts and 10 chats per day. [SuperClub members](https://cynicalsally.com/superclub) get unlimited everything — and Sally remembers you fully.

```
sally login your@email.com
```
